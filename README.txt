Código del Entregable 3 de la asignatura SECO

Para ejecutar el código en Python de los archivos seco_controller_*.py es necesario tener instaladas las siguientes dependencias en el sistema:
	* Python >= 3.9
	* Pipenv >= 2022.5.2

Para instalar Pipenv es necesario seguir las instrucciones en https://pypi.org/project/pipenv/

Seguidamente ejecutar
	$ pipenv install

Entonces se pueden ejecutar los ficheros de la siguiente manera
	$ pipenv run python {filename}.py
